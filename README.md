# Funcionalidade

O Boreal Chain é um projeto de estudo pessoal que possibilita a consulta de valores de criptomoedas em tempo real.

A barra de pesquisa (localizada no topo) funciona de forma simplificada que, ao digitar um valor, realiza uma pesquisa pelo nome da criptomoeda.

<br/>

Observações:

- Ao clicar na barra de pesquisa com um valor preenchido, ela automaticamente irá resetar a pesquisa, iniciando assim uma nova requisição;
- Ao entrar no projeto pela primeira vez, ou ao realizar uma consulta vazia, ele reune as 10 principais moedas do mercado.
